# 📋 最近请求记录 (Recent Request Log)

一个 SillyTavern 插件 —— 静默抓取每次发送给 AI 的完整提示词，方便随时回溯查看。

---

## ✨ 功能简介

- **自动抓取**：无需操作或打开面板，发送消息时自动记录发送给 AI 的完整提示词，包括系统提示、对话历史、用户输入等。
- **按角色分组**：每条记录按 `System` / `User` / `Assistant` 分类展示。
- **一键复制**：支持复制单条消息或整组记录的全部内容。
- **折叠管理**：记录默认折叠，点击展开；每条消息也可单独展开/收起。
- **明亮 / 暗色模式**：一键切换，设置会被浏览器记住。
- **移动 + 缩放**：点击标题栏空白处拖动位置，拖拽窗口右下角调整大小。
- **轻量运行**：所有数据仅保存在内存中，刷新或关闭页面后自动清空。

---

## 🚀 使用方法

### 安装

点击顶部 `扩展程序` 菜单 -> `安装扩展程序` -> 复制粘贴以下网址 -> `给所有人安装` -> 成功后刷新 SillyTavern 页面

```
SillyTavern-release/public/scripts/extensions/third-party/
```

---

### 面板入口

左下角 `扩展程序（魔法棒）` -> `「📘最近请求记录」`

---

### 功能说明

| 按钮 / 区域                   | 功能                                                                               |
| ------------------------- | -------------------------------------------------------------------------------- |
| **标题栏文字** `最近请求记录 (N/10)` | 显示当前已保存记录的数量。点击可折叠/展开整个面板。                                                       |
| **⚡ 广播监听**                | 开启后额外拦截浏览器原生 `fetch` 请求，尝试捕获部分绕过 SillyTavern 直接发送的 AI 请求。默认关闭，部分第三方插件无法记录时可尝试开启。 |
| **⏻ 总开关**                 | 绿色为开启。关闭后不再抓取任何新请求，也不会拦截 `fetch`。                                                |
| **🗑 清空**                 | 删除当前所有记录。                                                                        |
| **☀️ / 🌙 主题切换**          | 切换明亮模式与暗色模式。                                                                     |

## ⚠️ 注意事项

### 已知限制

- **Token 数为估算值**：面板显示的 Token 数量为粗略估算，并非 AI 实际计费使用的精确数值，**误差非常大**，仅供大致参考。
- **最多保存 10 条**：超出上限后最早记录会被自动覆盖，顶部为最新。
- **数据仅在内存中**：刷新、关闭标签页或重启 SillyTavern 后，所有记录都会被清除。
- **广播监听并非万能**：即使开启「广播监听」开关，部分第三方插件采用特殊方式发送的请求可能仍无法被捕获。
- **仅捕获请求**：本插件只记录**发送给 AI 的请求内容**（提示词），不记录 AI 返回的文本。

### 性能说明

- 插件对 SillyTavern 的正常使用几乎无性能影响，所有处理均为轻量级纯内存操作。
- 「广播监听」开启后会拦截浏览器的全局 `fetch` 调用，但仅分析与 AI 相关的请求体，不影响其他网络请求的响应速度。

### 隐私提示

- 所有数据仅存在于当前浏览器标签页的内存空间中，不会被上传、持久化存储或发送至任何外部服务。
- 切勿在公共设备上开启敏感对话后忘记关闭面板，面板内容对当前使用者完全可见。
