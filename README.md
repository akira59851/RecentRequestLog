# 📋 最近请求记录

一个SillyTavern 插件，自动抓取每次发送给 AI 的完整提示词，方便随时回溯查看。


## ✨ 功能简介

- **自动抓取**：无需打开面板，自动记录。
- **识别请求来源**：区分「原生」与「插件」。
- **按角色分组**：每条记录按 `System` / `User` / `Assistant` 分类展示。
- **一键复制**：支持复制单条消息或整组记录。
- **折叠管理**：消息和整个插件面板均能通过点击单独展开 / 收起。
- **移动 + 缩放**：点击标题栏空白处拖动；拖拽窗口右下角调整大小。
- **明亮 / 暗色模式**：一键切换，持久生效。


## 🚀 使用方法

### 安装

顶部菜单栏 `扩展程序` → `安装扩展程序` → 复制粘贴以下网址 → `给所有人安装` → 成功后刷新 SillyTavern 页面

```
https://github.com/akira59851/RecentRequestLog
```

### 面板入口

左下角 `扩展程序（魔法棒）` → `「📘最近请求记录」`

### 标题栏说明

- **最近请求记录 (N/10)**：显示当前已保存记录的数量。点击文字可折叠/展开整个面板。
- **⏻**：总开关，绿色为开启。关闭后不再记录任何新请求。
- **🗑**：清空当前所有记录。
- **☀️ / 🌙**：切换明亮 / 暗色主题。


## ⚠️ 注意事项

### 已知限制
- **Token 数为粗略估算**：面板显示的 Token 数量并非 AI 实际计费使用的精确数值，仅供大致参考。
- **最多保存 10 条**：超出上限后，最早记录会被自动覆盖，顶部为最新。
- **少数插件无法记录**：可以记录大部分插件的请求，但无法保证所有插件均能记录。

### 性能说明

- 几乎无性能影响，所有处理均为轻量级纯内存操作。
- 插件通过拦截浏览器全局 `fetch` 调用来捕获请求内容，仅分析与 AI 相关的请求体，不影响其他网络请求的正常响应。

### 隐私提示

- 所有数据仅存在于当前浏览器标签页的内存空间中，不会被上传、持久化存储或发送至任何外部服务。
- 刷新、关闭标签页或重启 SillyTavern 后，所有记录都会被清除。
